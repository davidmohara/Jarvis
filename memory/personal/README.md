# Personal Memory

User configuration, preferences, and identity context. Permanent. Always loaded at boot.
These files are **never promoted** to semantic — they are owned by David, not the dream cycle.

## Files
| File | Content |
|------|---------|
| `org.md` | Organizational structure and reporting relationships |
| `principles.md` | Core principles and decision-making rules |
| `quarterly-objectives.md` | Current quarter rocks and objectives |
| `vision.md` | Long-term vision and Lifebook context |
| `PREFERENCES.md` | Agent behavior preferences and customization |

## Schema
```yaml
---
type: personal
subject: "Description"
last-updated: YYYY-MM-DD
promoted: never
---
```
