# Tiling WM Cheat Sheet

**Modifier key: Caps Lock** (remapped to Ctrl+Alt)

## Navigation

| Action | Shortcut |
|--------|----------|
| Focus left / down / up / right | CapsLock + h / j / k / l |
| Move window left / down / up / right | CapsLock + Shift + h / j / k / l |
| Switch to last workspace | CapsLock + Tab |
| Move workspace to next monitor | CapsLock + Shift + Tab |

## Workspaces

| Action | Shortcut |
|--------|----------|
| Switch to workspace 1-9 | CapsLock + 1-9 |
| Move window to workspace 1-9 | CapsLock + Shift + 1-9 |
| Switch to named workspace | CapsLock + letter (e/m/c/b/s/z) |
| Move window to named workspace | CapsLock + Shift + letter |

## Windows

| Action | Shortcut |
|--------|----------|
| New terminal (WezTerm) | CapsLock + Enter |
| Close window | CapsLock + w |
| Fullscreen toggle | CapsLock + f |
| Resize smaller / larger | CapsLock + - / = |
| Toggle tiles layout | CapsLock + / |
| Toggle accordion layout | CapsLock + , |
| Join with adjacent window | CapsLock + Cmd + h/j/k/l |

## Modes

| Action | Shortcut |
|--------|----------|
| Enter app launcher mode | CapsLock + a |
| Enter service mode | CapsLock + Shift + ; |
| Reload config | CapsLock + Esc |

## App Launcher Mode (CapsLock + a, then...)

| Key | App |
|-----|-----|
| m | Messages |
| w | WhatsApp |
| s | Slack |
| t | Telegram |
| d | Discord |
| c | Calendar |

## Service Mode (CapsLock + Shift + ;, then...)

| Key | Action |
|-----|--------|
| r | Reset/flatten workspace layout |
| f | Toggle floating/tiling |
| Up/Down | Volume up/down |
| Shift + Down | Mute |
| Esc | Exit service mode |
