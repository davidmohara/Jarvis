**2. What you receive after the one-week discovery sprint**

- Time and materials estimate for the build phase

**3. Our capability to deliver instant output, online, via private domain or API**

- Private-domain hosted web application: brokers access via URL on a Nexben-branded domain, no install required
- API integration: if your platform has or can expose an API, we connect to it; if it doesn't exist yet, we can design the spec during discovery so your IT vendor can build it on their side

**5. Your high-level output list vs. stakeholder interviews - which approach and why**
